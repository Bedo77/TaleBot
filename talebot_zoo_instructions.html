TaleBot Online Practice: Enhanced Zoo Visit!

Welcome to the upgraded virtual Zoo Visit! Now featuring a 3D TaleBot and more commands. Your mission is to program TaleBot to visit the Panda.

Objective:

Program the 3D TaleBot to navigate the zoo map from its starting position to the Panda's enclosure.

The Virtual Zoo Map:

You'll be programming TaleBot on a vibrant zoo map like the one below. TaleBot will start near the bottom-left. The Panda is located towards the upper-middle area.

(The actual interactive map is in the environment)

How to Program Your 3D TaleBot:

In the Zoo Visit Environment, you'll find:

  • A 3D model of TaleBot on the map.
  • Command buttons to build your program.

Available Commands:

  • Movement: Move Forward, Move Backward, Turn Left (90°), Turn Right (90°).
  • Program Control:
    • Go!: Executes the current program sequence.
    • Clear All: Deletes the entire program.
    • Delete Last: Removes the last command added.
  • Advanced Commands:
    • Repeat Program + / -: Set how many times the entire program sequence will repeat.
    • Record Voice: Simulate recording a short voice message (you'll type it in).
    • Add 'Play Recording' to Program: Adds a command to play your recorded message.
    • Random Dance: Adds a command for TaleBot to perform a conceptual dance.

Programming Steps:

  1. Click the command buttons in the desired sequence. They will appear in the "Program Sequence" area.
  2. TaleBot starts facing "North" (towards the top of the map initially).
  3. Use the Repeat Program buttons if you want the whole sequence to run multiple times.
  4. To reach the Panda (estimate its position visually on the map in the environment):
    • You'll likely need a combination of Move Forward and Turn Left/Right commands.
    • Example: Move Forward, Move Forward, Turn Right, Move Forward... (This is just an example, you'll need to figure out the exact path!)
  5. Once your program is ready, click the Go! button.
  6. If TaleBot reaches the Panda, you'll get a success message and a little celebration! If not, use Delete Last or Clear All and try again.
